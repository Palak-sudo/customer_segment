# Customer Segmentation Analysis Project

## Overview
This project focuses on customer segmentation analysis using RFM (Recency, Frequency, Monetary) metrics and clustering techniques to identify different customer groups. The analysis helps businesses understand customer behavior and create targeted marketing strategies for better customer retention and increased revenue.

---

## Objective
The main objectives of this project are to:

- Analyze customer purchasing behavior
- Perform RFM analysis
- Segment customers into meaningful groups
- Identify high-value and low-value customers
- Recommend targeted marketing strategies

---

## Dataset Overview
The dataset contains customer purchase behavior information, including:

- Recency → How recently a customer made a purchase
- Frequency → How often a customer makes purchases
- Monetary → Total spending by the customer

These metrics are used to identify customer segments and purchasing patterns.

---

## Analysis Process
The following steps were performed during the analysis:

- Data cleaning and preprocessing
- RFM metric calculation
- Customer behavior analysis
- K-Means clustering for segmentation
- Cluster profiling and interpretation
- Visualization of customer groups

---

## Customer Segmentation Technique

### RFM Analysis
RFM analysis was used to evaluate customer value based on:

- Recency
- Frequency
- Monetary Value

### K-Means Clustering
K-Means clustering was applied to group customers with similar purchasing behavior into different segments.
